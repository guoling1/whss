钱包加加使用帮助

Q：顾客支付成功后，怎么查询到账？ A：您可以点击账单查看收款记录，也可以打开通知，收款成功后，钱包加加APP将会为您播报收款成功的语音提示。
Q：收款资金何时可以提取出来？ A：收款资金将在下一个工作日结算到您的账户余额，节假日产生的收款，将在假期结束的第一个工作日结算到您的账户余额。
Q：余额提现有什么要求？ A：余额随时可以提现，单笔提现金额不得低于10元。
Q：余额提现有手续费吗？ A：没有。
Q：余额什么时候能提取资金到我的银行卡？ A：提现申请成功后，2小时内到您注册商户时绑定的银行账户，请注意查收。
Q：顾客可以使用信用卡支付吗？ A：支持，顾客微信或支付宝中的余额与榜单的银行卡都可以进行支付。
Q：钱包加加支持多个店员吗？ A：支持，绑定店员后，店员与您都能收到顾客付款成功的通知。
