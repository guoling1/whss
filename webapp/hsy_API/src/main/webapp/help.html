钱包加加使用帮助

Q：顾客支付成功后，怎么查询到账？ A：您可以点击账单查看收款记录，也可以打开通知，收款成功后，钱包加加APP将会为您播报收款成功的语音提示。
Q：收款资金何时可以提取出来？ A：收款资金将在下一个工作日结算到您的账户余额，节假日产生的收款，将在假期结束的第一个工作日进行结算。
Q：顾客可以使用信用卡支付吗？ A：支持，顾客可以使用微信、支付宝账户中的余额或绑定的银行卡进行支付。
Q：我手机怎么听不到收款提示音： A：您可以通过“播报锦囊”功能来检查手机设置是否开启相关权限。
Q：我还需要多张收款码贴在店里的不同位置怎么办？ A：可通过APP端首页“店铺二维码”或“店铺管理”中查看并下载本店的电子收款码，下载后可自行打印。
Q：我有多个店铺需要收款，支持吗？ A：支持，您可以在店铺管理中创建多个店铺，分别绑定收款码可分开管理。
Q：钱包加加支持多个店员吗？ A：支持，绑定店员后，店员与您都能收到顾客付款成功的通知。
