钱包加加商户合作协议

商户（合同签署页甲方落款处签章的主体，以下称“甲方”）与北京金开门信息技术有限公司（以下称“乙方”）经友好协商，本着平等、自愿、诚信、互利的合作原则，根据《中华人民共和国合同法》等国家法律，法规规定，双方就乙方向甲方提供钱包加加业务服务事宜签订本协议，并承诺共同遵守。

1. 甲方责任和义务

1.1 遵守国家机关和行业管理机构制定的相关法律，法规和行业规范，接受乙方业务受理培训，知道和监督。 1.2 甲方承诺向乙方提供资料信息，包括但不限于工商登记资料，营业执照，法定代表人或负责人身份证复印件、银行卡复印件等真实、合法、有效、并承担因信息和身份资料不准确，不真实、不及时和不完整而引发的一切责任。 1.3 当甲方提供的信息发生变更时，至少应自变更之日起5个工作日内书面通知乙方，并提供相关资料。如因甲方未及时履行上述责任而造成的损失，由甲方自行承担。 1.4 甲方发生业务承包或转让时，未经乙方业务系统注册的商户账户和密码，因甲方商户账户密码保管及使用不当造成的经济损失，由甲方自行承担。 1.5 甲方发生业务承包和转让时，未经乙方书面同意，不得将本协议约定的钱包加加（钱包加加为北京金开门信息技术有限公司旗下的扫码收款业务的品牌名）业务转让或委托给第三方。 1.6 甲方承诺积极配合乙方处理因为系统升级，网络故障等原因导致的计算数据出错等，如出现多结算现象，甲方同意在后续结算款中扣除乙方多结的款项。 1.7 甲方有义务及时，妥善处理与甲方用户之间的投诉和纠纷，并承担全部法律责任，并不得因此而影响乙方的声誉和形象或给乙方造成经济损失。 1.8 甲方有权获得钱包加加相关的技术咨询和服务，并有义务及时乙方系统平台上的交易数据并存档，如对交易的账户处理有疑问，有向乙方提出查询的权利。 1.9 当交易存在诈骗、被盗用、洗钱、偷税等涉嫌违法犯罪行为时，甲方有义务随时配合、协助乙方或其他司法、职能部门调查可疑交易、冻结可疑账户，提供完整的客户资料和交易数据。

2. 乙方责任和义务

2.1 乙方有权审核甲方钱包加加业务用途，对不符合国家金融监管机构条例及其他制度规定的，非法的业务，乙方有权拒绝执行操作，并有权采用一切合法手段追究甲方的经济、法律责任。 2.2 乙方有权根据情况决定对钱包加加的产品功能和服务进行修改、升级，并暂停向甲方提供系统服务，但有义务提前告知甲方，并预告恢复日期。 2.3 乙方有合理理由认定交易存在可疑之处，或有合理根据怀疑甲方违反本协议，有违规交易或欺诈交易行为时，乙方有权采取暂缓甲方全部或部分交易结算、中止甲方钱包加加业务受理等措施。如甲方涉及套现、违规交易，乙方可冻结持有或划扣甲方款项，直至乙方认为甲方的欺诈交易对除甲方外的任何单位或个人无造成损失的可能。乙方拥有“违规交易”最终解释权。 2.4 在支付交易成功后，若出现支付人拒付以及其他无法确定责任人事故造成的争议时，乙方通知甲方，由甲方协调解决；当甲方与客户无法取得一致意见，无法协调解决时，若该争议或事故起因于甲方过错的，则从甲方的清算账户中扣除该拒付款项。 2.5 对于本协议有效期内出现的交易纠纷，甲方应在7个工作日内有效处理完毕，如不能处理完毕，则乙方有权对支付人提出的退款请求（在甲方账户余额足够的情况下）进行退款处理，并解除本协议。 2.6 乙方有义务对经查实的交易差错或乙方需调整的账务，负责按照差错争议处理的有关规定进行相应处理。 2.7 乙方有义务按协议规定及时将结算资金划入甲方在本协议指定的银行账户，并有权按照本协议的规定收取各项服务费用。 2.8 乙方有权获取跟甲方账单支付相关的交易清单信息，但乙方有义务保密。

3. 保密条款

3.1 在未经协议双方书面同意前，任何一方不得就本协议的主要内容向协议方以外的第三方公布。 3.2 根据本协议，双方的市场资料和其他商业资料，对协议方以外的第三方保密。 3.3 保密条款在本协议终结后五年内仍然有效。

4. 协议变更和解除

4.1 除本协议及其附件中已另有规定外，任何一方如因未能完全履行其在本协议项下的责任和义务，导致本协议无法继续履行或继续履行没有实际意义时，另一方有权解除本合作协议。 4.2 甲方出现下列任一种情形时，乙方有权立即终止甲方钱包加加业务，单方面无条件终止本协议，由此给乙方造成的相关损失和风险责任由甲方承担。乙方有权根据风险管理规定，使用甲方的相关风险信息。 4.2.1 虚假申请：以虚假资料或冒用其他商户资料向乙方申请为签约商户 4.2.2 套现：甲方和/或甲方员工或其他第三方勾结，或甲方和/或甲方员工利用钱包加加业务以虚假交易套取交易补贴及奖励； 4.2.3 套取交易补贴：甲方和/或甲方员工与用户或其他第三方勾结，或甲方和/或甲方员工利用钱包加加业务以虚假交易套取交易补贴及奖励； 4.3 甲方业务变更，终止或主体注销未及时通知乙方，乙方在联系无反馈的情况下，有权中止或终止本协议。 4.4 乙方根据业务整体运作、风险管理、外部环境等情况，享有选择暂停、终止本协议全部或者部分产品服务之权利，而不承担违约责任。 4.5 协议解除或终止后，甲方有义务对合作期内发生交易的支付人提出的退款请求及时进行处理，乙方保留直接用甲方账户资金向支付人退款的权利，并有权协助或代表支付人向甲方追偿退款。

5. 免责条款

5.1 若遇国家行政管理部门政策调整，或中国人民银行、乙方的合作银行、合作机构、合作伙伴的相关政策调整的，调整方应及时通知另一方，由双方协商议定，如协商不能达成一致的，被要求调整方有权终止本协议。 5.2 对于不可抗力或乙方不能控制的原因造成的服务中断或其他缺陷，乙方不承担相关责任，但将尽力减少因此而给甲方造成的损失和影响。 5.3 因第三方系统故障引起的双方业务系统暂时性不能或部分不能提供钱包加加业务的，甲乙双方免责。

6. 其他

6.1 乙方向甲方提供钱包加加业务服务，对于甲方与消费者在出售商品的质量，数量或提供的服务上有任何争议、投诉或其他纠纷，应由甲方与消费者出售商品的质量、数量或提供的服务上有任何争议，投诉或其他纠纷，应由甲方与消费者直接协商解决，乙方不承担任何责任。 6.2 除本协议及其附件中已另有规定者外，任何一方如因未能完全履行其在本协议项下的责任和义务，造成其他损失的，应负责赔偿其他方因此产生的直接损失，损害，并承担由此造成的责任和/或开支。 6.3 本协议有效期一年，如双方无争议及条款变更，自动续期。 6.4 如双方在协议执行中发生争议，应首先本着友好协商的精神商议解决，如协商不能解决时，任何一方均可向乙方所在地人民法院诉讼解决。 6.5 本协议修改、补充、变更、必须经过甲、乙双方协商一致，并经双方签署书面协议后方能生效，并作为本协议，与本协议具有相同的法律效力。
